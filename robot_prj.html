When AI, Robotics, and Security meet... this happens.

In this project, we built a robot that could autonomously drive itself, use computer vision to detect dangerous objects and violent activities, and send those incidents to a server that we built. Those incidents can then be viewed by the user via the GUI in our webapp.

I was the full-stack developer / cybersecurity engineer / network engineer of this project. That means I was in charge of developing the server, the web app, the secure communication between the server and the robot, and the security both the server and the robot... all in just FOUR months.
